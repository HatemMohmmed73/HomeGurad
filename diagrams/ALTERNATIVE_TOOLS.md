# Alternative Tools for Creating UML Diagrams

This document lists alternative websites and tools similar to PlantUML for creating UML diagrams. All tools are organized by type and include pros/cons.

## üåê Free Web-Based Tools (No Installation Required)

### 1. **Draw.io / diagrams.net** ‚≠ê RECOMMENDED
- **Website**: https://app.diagrams.net/ or https://www.diagrams.net/
- **Best For**: All diagram types, professional appearance
- **Features**:
  - Completely free, no account required
  - Works offline (can download desktop version)
  - Extensive shape libraries (UML, network, flowcharts)
  - Export to PNG, SVG, PDF, HTML
  - Integrates with Google Drive, OneDrive, GitHub
  - Real-time collaboration
- **Pros**: 
  - Very user-friendly interface
  - Professional output quality
  - No registration needed
  - Can save locally or to cloud
- **Cons**: 
  - Manual diagram creation (not code-based)
  - Can be slower for complex diagrams

### 2. **Mermaid Live Editor**
- **Website**: https://mermaid.live/
- **Best For**: Quick diagrams, markdown integration
- **Features**:
  - Code-based diagram creation (like PlantUML)
  - Syntax similar to PlantUML
  - Renders instantly
  - Export to PNG, SVG
  - Works in GitHub/GitLab markdown
- **Pros**:
  - Fast and lightweight
  - Version control friendly (text-based)
  - Good for documentation
- **Cons**:
  - Limited diagram types compared to PlantUML
  - Less customization options

### 3. **Lucidchart**
- **Website**: https://www.lucidchart.com/
- **Best For**: Professional diagrams, team collaboration
- **Features**:
  - Free tier available (limited)
  - UML templates
  - Real-time collaboration
  - Integration with Google Workspace, Microsoft Office
  - Export to multiple formats
- **Pros**:
  - Professional templates
  - Good collaboration features
  - Clean interface
- **Cons**:
  - Free tier has limitations (3 documents)
  - Requires account registration
  - Some features are paid-only

### 4. **Creately**
- **Website**: https://creately.com/
- **Best For**: Team collaboration, templates
- **Features**:
  - Free tier with 3 documents
  - UML diagram templates
  - Real-time collaboration
  - Export options
- **Pros**:
  - Good templates library
  - Easy to use
- **Cons**:
  - Limited free tier
  - Requires registration

### 5. **Gliffy**
- **Website**: https://www.gliffy.com/
- **Best For**: Confluence integration
- **Features**:
  - Free trial available
  - UML support
  - Confluence integration
- **Pros**:
  - Good for Atlassian users
- **Cons**:
  - Limited free version
  - Requires account

## üöÄ Quick Start Guides

### Draw.io Quick Start:
1. Go to https://app.diagrams.net/
2. Choose "Create New Diagram"
3. Select "Blank Diagram" or "UML" template
4. Use shape library on left (search "UML")
5. Drag and drop shapes, connect with arrows
6. Export: File ‚Üí Export as ‚Üí PNG/PDF/SVG

### Mermaid Quick Start:
1. Go to https://mermaid.live/
2. Paste Mermaid code (from `.mmd` files)
3. Diagram renders automatically
4. Click "Actions" ‚Üí "Download PNG/SVG"

### PlantText Quick Start:
1. Go to https://www.planttext.com/
2. Paste PlantUML code (from `.puml` files)
3. Diagram renders automatically
4. Right-click ‚Üí "Save Image As"

## üìù Tips for Each Tool

### Draw.io:
- Use "Arrange" ‚Üí "Layout" for automatic alignment
- Enable "View" ‚Üí "Grid" for better alignment
- Use layers for complex diagrams
- Save frequently (auto-save available)

### Mermaid:
- Use `%%{init: {'theme':'base'}}%%` for themes
- Comments: `%% This is a comment`
- Supports markdown in labels

### PlantText:
- Same syntax as PlantUML
- Use `@startuml` and `@enduml` tags
- Supports all PlantUML features
