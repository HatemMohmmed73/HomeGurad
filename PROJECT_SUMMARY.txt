The HomeGuard project has successfully implemented a comprehensive smart home 
security gateway solution . The system consists of a FastAPI-based backend API with 
15 essential REST endpoints for device 
and alert management, integrated with MongoDB for authentication and system settings. The 
frontend is a modern React dashboard built with TypeScript and Tailwind CSS, 
providing real-time device monitoring, alert management, and system configuration. 
The solution features JSON file-based data integration that automatically 
synchronizes with Orange Pi monitoring scripts, enabling seamless device and alert 
data updates. The entire application is containerized using Docker with an 
all-in-one image approach, deployed via Docker Compose for both development and 
production environments. The system includes WebSocket support for real-time 
notifications and automatic polling mechanisms (5-second intervals) to ensure the 
dashboard reflects the latest data from the Orange Pi. All core software 
components including JWT-based authentication, device management API, alert system, 
and deployment infrastructure are fully functional and production-ready.
